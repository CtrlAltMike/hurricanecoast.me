Font options

Four alternatives to the current Fraunces + Inter pairing. Each mockup shows the domain name in the nav, a hero-scale heading, a section heading, lede, checklist, and buttons — the full range where font choice is felt.

Option 1

Newsreader

Newspaper-quality Google serif. The brand name reads like a masthead — specific, grounded, no optical tricks. Strong bold at large sizes with the kind of authority you'd expect from a publication that gets things right. Keeps Inter for body.

Newsreader 600/700 + Inter 400/600 Google Fonts Variable optical size

Option 2

Instrument Serif

Clean, contemporary Google serif from 2023. Single weight — no bold — so large headings rely on size and letterform rather than stroke weight. Works consistently from nav scale to hero scale without Fraunces's optical quirkiness. Reads younger and more coastal-modern.

Instrument Serif 400 + Inter 400/600 Google Fonts Single weight — size does the work

Option 3

Libre Baskerville

Traditional transitional serif with a heavy, confident bold. The brand name looks like something you'd trust with a checklist — capable and utilitarian rather than editorial. Large headings have solid weight. Most grounded of the four; least precious.

Libre Baskerville 400/700 + Inter 400/600 Google Fonts Strong bold — weight does the work

Option 4

Inter brand · DM Serif Display headings

Structural split: the domain name moves to Inter semibold — the same font as body text — so it's never asked to perform at a size it wasn't designed for. Display headings get DM Serif Display, a serif built exclusively for large sizes. Cleanest architectural separation of the four.

Inter 400/600 (brand + body) · DM Serif Display 400 (headings) Google Fonts Nav sans · headings serif
